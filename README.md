## ⚡ Ключевые особенности (Features)

* **Terminal Gateway (Landing)** — Входной экран в стиле кибернетического шлюза. Поддерживает протокол `[AUTO_BYPASS]` (эмуляция хакерского перебора токена доступа) и интерактивный выбор региональных кластеров.
* **Live System Logs** — Полноценная консоль в нижней части экрана, имитирующая реалтайм обработку данных. Каждые несколько секунд генерирует новые логи с умным авто-скроллом вниз.
* **Dynamic Metrics Simulation** — Аппаратные показатели модулей (выделение памяти, пинг) колеблются на лету, создавая ощущение подключенного живого сервера.
* **Strict Engineering Design** — Интерфейс в стиле IDE (LazyVim/VSCode) и технической документации с моноширинными акцентами (`JetBrains Mono`). Максимальная информативность без визуального шума.

## 🛠️ Стек технологий (Tech Stack)

* **Core:** React 18+ (SPA)
* **State Management:** React Hooks (`useState`, `useEffect`, `useRef`)
* **Styling:** Pure CSS3 (Custom Properties / CSS Variables)
* **Fonts:** JetBrains Mono & Inter via Google Fonts
* **Data Layer:** Local JSON Scheme

## 🚀 Быстрый запуск (Local Setup)

Чтобы запустить проект локально на своем компьютере, выполните три простые команды в терминале:

1. **Клонирование репозитория:**
   ```bash
   git clone https://github.com<ваш_логин>/<имя_репозитория>.git
   cd <имя_репозитория>
   ```

2. **Установка зависимостей:**
   ```bash
   npm install
   ```

3. **Запуск сервера разработки:**
   ```bash
   npm run dev
   ```

После этого откройте адрес `http://localhost:5173` в вашем браузере.
